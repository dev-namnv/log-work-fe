# Hướng dẫn định nghĩa Types

## Quy tắc

- Dùng `interface` cho domain object types.
- Dùng `type` cho union types, computed/utility types.
- `app/types/index.ts` re-export tất cả types từ các sub-files.
- Không import từ các sub-files trực tiếp trong app — chỉ import từ `~/types`.

## Cấu trúc file

```
app/types/
  index.ts          # Re-exports: export * from './organization'; …
  organization.ts   # Organization, OrganizationSettings
  employee.ts       # Employee
  work-log.ts       # WorkLog, WorkDaySummary, WorkDayStatus
  reports.ts        # PayrollPeriod, EmployeePayroll
```

## Ví dụ `index.ts`

```ts
export * from './organization';
export * from './employee';
export * from './work-log';
export * from './reports';
```

## Lưu ý

- Tất cả date fields dùng `string` (`YYYY-MM-DD`), không dùng `Date`.
- Tất cả time fields dùng `string` (`HH:mm`), không dùng `Date` hay `number`.
- `WorkDaySummary` là kiểu **computed** — không lưu vào DB, không persist.
- Timestamp fields (`createdAt`, `updatedAt`) dùng ISO string từ server.

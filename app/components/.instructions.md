# Hướng dẫn viết Components

## Quy tắc chung

- Mỗi component một file, export default.
- Props interface đặt trong cùng file, đặt tên `<ComponentName>Props`.
- Không dùng inline styles; chỉ dùng Tailwind utility classes.
- Dùng `cn()` (clsx + tailwind-merge) để merge className có điều kiện.

```ts
// app/lib/utils.ts
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';
export function cn(...inputs: ClassValue[]) {
	return twMerge(clsx(inputs));
}
```

## `components/ui/` — Primitive components

Các component dùng nhiều nơi, không chứa business logic.

### Button

```tsx
interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
	variant?: 'primary' | 'secondary' | 'danger' | 'ghost';
	size?: 'sm' | 'md' | 'lg';
	loading?: boolean;
}
```

### Badge — hiển thị WorkDayStatus

```tsx
interface BadgeProps {
	status: WorkDayStatus;
}
// Màu sắc theo status:
// present    → green
// absent     → red
// late       → yellow
// early-leave → orange
// overtime   → blue
// half-day   → purple
// holiday    → gray
// weekend    → gray
```

### WorkTimeInput

```tsx
// Input "HH:mm" với validation
interface WorkTimeInputProps {
	label: string;
	value: string | null;
	onChange: (value: string | null) => void;
	error?: string;
}
```

## `components/forms/` — Form components

### WorkLogForm

Form tạo / sửa bản ghi chấm công.

```tsx
interface WorkLogFormProps {
	defaultValues?: Partial<WorkLog>;
	employees: Employee[];
	errors?: Record<string, string[]>; // Từ action validation
}
// Dùng React Router <Form method="post"> hoặc useFetcher().Form
```

### OrganizationSettingsForm

Form cấu hình giờ làm của cơ quan.

```tsx
interface OrganizationSettingsFormProps {
	settings: OrganizationSettings;
	errors?: Record<string, string[]>;
}
// Hiển thị time picker cho: workStartTime, workEndTime, lunchBreakStart, lunchBreakEnd
// Number input cho: standardWorkHoursPerDay, allowLateMinutes, allowEarlyLeaveMinutes
// Checkbox group cho: workDays (Thứ 2 → Chủ nhật)
```

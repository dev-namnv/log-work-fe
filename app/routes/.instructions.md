# Hướng dẫn viết Route files

## Quy tắc bắt buộc

- Mọi route **phải** export `meta` function.
- Data fetching chỉ được thực hiện trong `loader` (server-side). **Không** dùng `useEffect` + `fetch`.
- Form mutations dùng `action` function + React Router `<Form>` component.
- Partial/background updates (ví dụ: toggle check-in) dùng `useFetcher`.

## Cấu trúc file route chuẩn


## Naming convention

### Quy tắc đặt tên file

- Mỗi module có **folder riêng** trong `app/routes/<module>/`.
- Tên file theo pattern: `<page>.<module>.tsx`.
- Dynamic param đặt ở đầu tên file: `$<param>.<page>.<module>.tsx`.
- Route index (trang danh sách) của module: `index.<module>.tsx`.

### Ví dụ

| Route                           | Folder / File                                         |
| ------------------------------- | ----------------------------------------------------- |
| `/`                             | `routes/home.tsx`                                     |
| `/auth/login`                   | `routes/auth/login.auth.tsx`                          |
| `/auth/register`                | `routes/auth/register.auth.tsx`                       |
| `/auth/forgot-password`         | `routes/auth/forgot-password.auth.tsx`                |
| `/auth/reset-password`          | `routes/auth/reset-password.auth.tsx`                 |
| `/auth/verify-otp`              | `routes/auth/verify-otp.auth.tsx`                     |
| `/auth/verify-email/:token`     | `routes/auth/$token.verify-email.auth.tsx`            |
| `/work-logs`                    | `routes/work-logs/index.work-logs.tsx`                |
| `/work-logs/:date`              | `routes/work-logs/$date.work-logs.tsx`                |
| `/employees`                    | `routes/employees/index.employees.tsx`                |
| `/employees/:id`                | `routes/employees/$id.employees.tsx`                  |
| `/organizations`                | `routes/organizations/index.organizations.tsx`        |
| `/organizations/:id`            | `routes/organizations/$id.organizations.tsx`          |
| `/organizations/:id/settings`   | `routes/organizations/$id.settings.organizations.tsx` |
| `/reports/attendance`           | `routes/reports/attendance.reports.tsx`               |
| `/reports/payroll`              | `routes/reports/payroll.reports.tsx`                  |
| `/reports/payroll/:year/:month` | `routes/reports/$year.$month.payroll.reports.tsx`     |

## Khai báo route trong `app/routes.ts`

```ts
import {
	type RouteConfig,
	index,
	route,
	layout,
	prefix,
} from '@react-router/dev/routes';

export default [
	index('routes/home.tsx'),

	// Auth — dùng AuthLayout riêng (không có header/footer chính)
	layout('components/layout/AuthLayout.tsx', [
		...prefix('auth', [
			route('login', 'routes/auth/login.auth.tsx'),
			route('register', 'routes/auth/register.auth.tsx'),
			route('forgot-password', 'routes/auth/forgot-password.auth.tsx'),
			route('reset-password', 'routes/auth/reset-password.auth.tsx'),
			route('verify-otp', 'routes/auth/verify-otp.auth.tsx'),
			route('verify-email/:token', 'routes/auth/$token.verify-email.auth.tsx'),
		]),
	]),

	route('work-logs', 'routes/work-logs/index.work-logs.tsx'),
	route('work-logs/:date', 'routes/work-logs/$date.work-logs.tsx'),

	route('employees', 'routes/employees/index.employees.tsx'),
	route('employees/:id', 'routes/employees/$id.employees.tsx'),

	route('organizations', 'routes/organizations/index.organizations.tsx'),
	route('organizations/:id', 'routes/organizations/$id.organizations.tsx'),
	route(
		'organizations/:id/settings',
		'routes/organizations/$id.settings.organizations.tsx',
	),

	...prefix('reports', [
		route('attendance', 'routes/reports/attendance.reports.tsx'),
		route('payroll', 'routes/reports/payroll.reports.tsx'),
		route(
			'payroll/:year/:month',
			'routes/reports/$year.$month.payroll.reports.tsx',
		),
	]),
] satisfies RouteConfig;
```

## Error handling

```tsx
// Trong loader — resource not found
throw new Response('Không tìm thấy nhân viên', { status: 404 });

// Trong action — validation error
import { data } from 'react-router';
return data({ errors: result.error.flatten() }, { status: 422 });
```

## Payroll route — lưu ý đặc biệt

- Logic tính lương **không** được viết trong loader. Loader chỉ tổng hợp dữ liệu thô.
- Gọi `buildPayrollPeriod()` từ `app/services/payroll.service.ts` để tính toán.
- `buildPayrollPeriod` trong service sẽ gọi các hàm từ `app/lib/calculations.ts`.
